# woflStrology

## Welcome to woflStrology - what is it?/disclaimer
woflStrology is just a silly li'l astrology/horoscope app to entertain and amuse.
Yes, it gives advice - but seriously, if you're dumb enough to take the advice of a horoscope program to heart to your actual detriment, you're probably too dumb to actually sue me or have the money to do so anyways.

Either way, **ALL ADVICE GIVEN IS FOR ENTERTAINMENT PURPOSES ONLY and ANY RESULTS  DERIVED FROM THE ACTUAL CONTENT RESULTING FROM RUNNING THIS SOFTWARE is ENTIRELY AT THE USERS OWN RISK.**


## Installing/Running:
Either select .zip for your OS and follow README contained within,

Or go to `/src/` and just run
```bash
Python woflstrology-vX.X.X
```


## Contributing
Want to add features or fix bugs? Here's how:

1. **Fork** this repo (click Fork button)
2. **Clone** your fork: `git clone https://github.com/YOUR_USERNAME/astrology.git`
3. **Create a branch**: `git checkout -b my-feature`
4. create working environment:
```bash
python -m venv venv
./venv/scripts/activate
py -3 -m pip install --upgrade -r requirements.txt
```
5. **Make changes** and test them
6. **Commit**: `git commit -m "Description of changes"`
7. **Push**: `git push origin my-feature`
8. **Create Pull Request** on GitHub

See [CONTRIBUTING.md](CONTRIBUTING.md) for detailed instructions.
