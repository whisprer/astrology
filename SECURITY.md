# Security Policy

## Reporting a Vulnerability
If you discover a potential vulnerability:
1. **Do not** open a public GitHub issue immediately.
2. Contact the maintainer privately via:
   - Email: security@whispr.dev
   - Or via GitHub’s “Report a vulnerability” feature.

We aim to respond within **72 hours**.

## Security Scope
This project focuses on *entertainment* and absolutely of nothing more, and to that end, operations of
woflStrology.py, woflStrology-win.exe, woflStrology-macOS and woflStrology-Linux
It **does not** claim saftey of computer hardware (especially peripherals or
cloud connected devices) or any other associated software.
